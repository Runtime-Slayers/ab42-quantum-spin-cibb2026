# Simulation Data Directory

This directory contains key numerical outputs and benchmark datasets reported in our CIBB 2026 manuscript.

## File Descriptions

- **`energy_convergence.csv`**: Optimization trajectory across 50 iterations comparing R-QNG with standard Adam optimizer on the Aβ₄₂ spin Hamiltonian landscape.
- **`rqng_vs_baselines.csv`**: Comprehensive performance benchmarks comparing R-QNG, Adam, Simulated Annealing (SA), and Genetic Algorithms (GA).
- **`heom_coherence.csv`**: radical-pair spin coherence decay dynamics comparing Markovian (Lindblad) vs. HEOM-MPO non-Markovian solvent models.
- **`isotope_energies.csv`**: Ground-state energy distributions (80 samples each) for ¹⁴N vs. ¹⁵N isotopic substitutions at the LVFFA core.
